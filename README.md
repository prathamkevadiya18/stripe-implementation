# Stripe Checkout Elements - Integration Guide

This guide provides step-by-step instructions on how to set up, run, and test the Stripe Checkout Elements integration. 

The architecture consists of:
1. **Frontend (Port 3001):** A static HTML workspace serving the Stripe payment form, success page, and failure page.
2. **Backend (Port 3002):** A NestJS server that creates Stripe Checkout Sessions, processes webhooks, and manages orders locally using a JSON database file (`orders.json`).

---

## 1. Prerequisites

Make sure you have the following installed on your machine:
* **Node.js** (v18 or higher recommended)
* **npm** (comes with Node)
* **Stripe CLI** (for local webhook testing)
* **Postman** (for testing REST API endpoints)

---

## 2. Setup & Configuration

### Backend Environment Variables
1. Navigate to the `backend` directory.
2. Create a `.env` file (or copy from `.env.example`):
   ```bash
   cd backend
   cp .env.example .env
   ```
3. Open `.env` and fill in your Stripe API keys:
   ```env
   STRIPE_SECRET_KEY=sk_test_your_secret_key_here
   STRIPE_PUBLISHABLE_KEY=pk_test_your_publishable_key_here
   STRIPE_API_VERSION=2026-03-25.dahlia
   STRIPE_WEBHOOK_SECRET=whsec_your_webhook_secret_here (will be generated by Stripe CLI)

   # Frontend/Website URL redirection
   FRONTEND_URL=http://localhost:3001
   WEBSITE_URL=http://localhost:3001
   ```

---

## 3. How to Run the Application

### Step 1: Start the Backend
1. In a terminal, navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the NestJS server in development mode:
   ```bash
   npm run start:dev
   ```
   *The backend will now be running at **`http://localhost:3002`**.*

### Step 2: Start the Frontend
1. Open a **new terminal** and navigate to the project root:
   ```bash
   cd E:\extra\stripe
   ```
2. Host the frontend static directory using `npx serve`:
   ```bash
   npx serve -l 3001 frontend
   ```
   *The frontend will now be running at **`http://localhost:3001`**.*

### Step 3: Run the Stripe Webhook Listener
To receive local webhook notifications from Stripe (for updating order statuses in `orders.json` when a payment succeeds or fails):
1. Open a **new terminal** and log in to Stripe:
   ```bash
   stripe login
   ```
2. Start forwarding webhooks to the local backend port `3002`:
   ```bash
   stripe listen --forward-to http://localhost:3002/payments/webhook
   ```
3. Copy the **webhook signing secret** printed by the Stripe CLI (it starts with `whsec_...`).
4. Paste it into the `STRIPE_WEBHOOK_SECRET` variable inside the backend `.env` file. Restart the backend server for it to apply.

## 4. Testing End-to-End with Postman / API Client

Follow these simple steps to generate a payment session and complete a test payment:

### Step A: Create a New Order in Postman
Send a request to create a new order record on your backend.
* **HTTP Method:** `POST`
* **URL:** `http://localhost:3002/orders`
* **Headers:** `Content-Type: application/json`
* **Request Body (JSON):**
  ```json
  {
    "total": 150.00,
    "currency": "USD"
  }
## OUTPUT 
* **Expected Response (JSON):**
  ```json
  {
    "id": "abc123xyz",
    "total": 150,
    "currency": "USD",
    "orderNumber": "ORD-1001",
    "status": "pending",
    "createdAt": "2026-06-03T11:54:33.000Z",
    "updatedAt": "2026-06-03T11:54:33.000Z"
  }
  ```
* **What to do:** Copy the **`id`** value from the response (e.g., `abc123xyz`).

### Step B: Generate Stripe Client Secret in Postman
Create a payment checkout session for the order ID you just copied.
* **HTTP Method:** `POST`
* **URL:** `http://localhost:3002/payments/create-session`
* **Headers:** `Content-Type: application/json`
* **Request Body (JSON):**
  ```json
  {
    "orderId": "abc123xyz"
  }
## OUTPUT
* **Expected Response (JSON):**
  ```
  {
    "clientSecret": "cs_test_a1b2c3d4_secret_...",
    "sessionId": "cs_test_a1b2c3d4..."
  }
  ```
* **What to do:** Copy the **`clientSecret`** value (starts with `cs_test_` and contains `_secret_`).

---

## 5. Complete the Payment on Frontend

1. Open your browser and navigate to:
   👉 **`http://localhost:3001`**
2. In the form, fill in the following details:
   * **Stripe Publishable Key:** Paste your test publishable key (starts with `pk_test_...`).
   * **Customer Email:** `customer@example.com`
   * **Client Secret:** Paste the **`clientSecret`** value that you copied from **Step B** in Postman.
3. Click the **Load Payment Form** button.
4. The secure Stripe checkout fields will load underneath the form. Enter the test card details:
   * **Card Number:** `4242 4242 4242 4242`
   * **Expiry Date:** Any future date (e.g., `12 / 29`)
   * **CVC:** `123`
5. Click **Pay Now**.
6. Stripe will process the card and automatically redirect you to the payment successful page (`success.html`).
7. Simultaneously, the Stripe Webhook listener will receive the successful payment event and update the order status on your backend to `"paid"`.

## 6. Alternative: Express.js Implementation

If you want to migrate or build the backend in **Express.js** instead of NestJS, here is a complete, copy-pasteable server code. 

### Prerequisites for Express
Install the required packages in your Express project:
```bash
npm install express stripe dotenv cors
```

### Express Server Code (`server.js`)
Create a file named `server.js` and paste the following code:

```javascript
const express = require('express');
const cors = require('cors');
const dotenv = require('dotenv');
const stripe = require('stripe');

// Load environment variables
dotenv.config();

const app = express();
const stripeClient = stripe(process.env.STRIPE_SECRET_KEY, {
  apiVersion: process.env.STRIPE_API_VERSION || '2026-03-25.dahlia'
});

// Enable CORS
app.use(cors({ origin: '*' }));

// ----------------------------------------------------
// 1. Stripe Webhook Endpoint (Requires RAW Body Parser)
// ----------------------------------------------------
app.post('/payments/webhook', express.raw({ type: 'application/json' }), async (req, res) => {
  const signature = req.headers['stripe-signature'];
  const webhookSecret = process.env.STRIPE_WEBHOOK_SECRET;

  if (!webhookSecret) {
    console.error('STRIPE_WEBHOOK_SECRET is not configured');
    return res.status(400).send('STRIPE_WEBHOOK_SECRET is not configured');
  }

  let event;
  try {
    // Construct event using the raw body buffer
    event = stripeClient.webhooks.constructEvent(req.body, signature, webhookSecret);
  } catch (err) {
    console.error(`Webhook Signature Verification Failed: ${err.message}`);
    return res.status(400).send(`Webhook Error: ${err.message}`);
  }

  console.log(`Received Stripe Event: ${event.type}`);

  // Handle successful/failed payments, dispute events, etc.
  switch (event.type) {
    case 'payment_intent.succeeded': {
      const pi = event.data.object;
      const orderId = pi.metadata?.orderId;
      console.log(`Payment Succeeded for orderId: ${orderId}, paymentIntentId: ${pi.id}`);
      // TODO: Update your database status to "paid" for this orderId
      break;
    }
    case 'payment_intent.payment_failed': {
      const pi = event.data.object;
      const orderId = pi.metadata?.orderId;
      console.log(`Payment Failed for orderId: ${orderId}, paymentIntentId: ${pi.id}`);
      // TODO: Update your database status to "failed" or "cancelled" for this orderId
      break;
    }
    case 'charge.refunded': {
      const charge = event.data.object;
      const orderId = charge.metadata?.orderId;
      console.log(`Charge Refunded for orderId: ${orderId}`);
      // TODO: Update order status to "refunded"
      break;
    }
    default:
      console.log(`Unhandled event type: ${event.type}`);
  }

  res.json({ received: true });
});

// ----------------------------------------------------
// Middleware for remaining JSON endpoints
// ----------------------------------------------------
app.use(express.json());

// Mock database (equivalent to orders.json)
const mockOrders = {
  "abc123xyz": { id: "abc123xyz", total: 150.00, currency: "USD", orderNumber: "ORD-1001" }
};

// ----------------------------------------------------
// 2. Create Stripe Checkout Session Endpoint
// ----------------------------------------------------
app.post('/payments/create-session', async (req, res) => {
  const { orderId } = req.body;
  
  if (!orderId) {
    return res.status(400).json({ error: 'orderId is required' });
  }

  // Retrieve order details
  const order = mockOrders[orderId];
  if (!order) {
    return res.status(404).json({ error: 'Order not found' });
  }

  const websiteUrl = process.env.WEBSITE_URL || 'http://localhost:3001';
  const amountInCents = Math.round(Number(order.total) * 100);

  try {
    // Create Stripe Checkout Session in Elements mode
    const session = await stripeClient.checkout.sessions.create({
      ui_mode: 'elements',
      mode: 'payment',
      payment_method_types: ['card', 'klarna'],
      line_items: [
        {
          price_data: {
            currency: order.currency.toLowerCase() || 'usd',
            product_data: {
              name: `Order #${order.orderNumber}`,
              description: `Payment for Order #${order.orderNumber}`,
            },
            unit_amount: amountInCents,
          },
          quantity: 1,
        },
      ],
      metadata: { orderId: order.id },
      payment_intent_data: { metadata: { orderId: order.id } },
      return_url: `${websiteUrl}/success.html?session_id={CHECKOUT_SESSION_ID}&order_id=${order.id}`,
    });

    res.json({
      clientSecret: session.client_secret,
      sessionId: session.id
    });
  } catch (error) {
    console.error(`Failed to create Checkout Session: ${error.message}`);
    res.status(500).json({ error: error.message });
  }
});

// ----------------------------------------------------
// 3. Get Session Details Endpoint (Mocked/Direct)
// ----------------------------------------------------
app.get('/payments/session/:id', (req, res) => {
  const sessionId = req.params.id;
  
  // Return preset mock session data directly
  res.json({
    sessionId: sessionId,
    status: 'complete',
    paymentStatus: 'paid',
    amountTotal: 15000,
    currency: 'usd',
    paymentIntentId: 'pi_mock_123456789',
    refunds: []
  });
});

// Start the server
const PORT = process.env.PORT || 3002;
app.listen(PORT, () => {
  console.log(`Express server running on port ${PORT}`);
});
```
